Conhecendo o elemento de navegação

Definição de link

Um link é um elemento que permite navegar de uma página, documento, recurso para outra.

No html, a gente usa os links para:

  • Abrir um site externo
  • Levar o usuário para outra página do site
  • abrir aplicativos como e-mail e telefone

Conhecendo elemento a

O elemento usado para criar links é a tag a (anchor)

href = hyperText Reference é o atributo usado para especificar o destino do link, ou seja, a URL ou o caminho para onde o link deve levar o usuário.

Target - é o atributo que especifica onde o link será aberto. Os valores mais comuns são:

  • _self - abre o link na mesma janela (padrão)
  • _blank - abre o link em uma nova janela ou aba
  • _parent - abre o link na janela pai
  • _top - abre o link na janela principal, eliminando quaisquer frames
  • frame/iframe abre o link em um frame ou iframe do pai
