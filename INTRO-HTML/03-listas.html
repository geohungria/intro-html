Conhecendo Listas HTML

As listas do html, servem para organizar informacoes relacionados de forma estruturada e semantica .

As listas sao comportas por elementos chamadas de LI.

Os Li (list item da lista). As listas nao possuim limites de imtens.

Ordered List - OL

O elemento OL serve para criarmos uma lista ordenada, ou seja, com ordem, importancia

Normalmente aparece numeradas

Unordered List - UL

O elemento UL serve para criarmos uma lista não ordenada, ou seja, sem ordem específica.

Normalmente aparece com marcadores (bullets)/bolinhas.

Exemplo rápido

  • trabalho
  • estudar
  • tomar banho
  • jogar
  • assistir TV

Curiosidade importante

você pode colocar listas dentro de outras listas.

description List - DL

O elemento DL serve para criarmos uma lista de definições, ou seja, uma lista onde cada item é uma definição de um termo.

Normalmente aparece com o termo em negrito e a definição em texto normal.
