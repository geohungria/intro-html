Conhecendo o elemento de Imagens

Sobre as imagens

Trabalhar com imagens no HTML envolve muito mais doque apenas colocar uma imagem na página.

Existem várias formas, técnicas e recusos para exibir, organizar, otimizar e adaptar as imagnes para diferentes dispositivos.

Formatos mais comuns de imagens

  • JPEG/JPG: Fotos e imagens compactadas
  • PNG: Imagens com transparência
  • WEBP: Formato moderno de imagem com compressão eficiente
  • GIF: Animações e imagens simples
  • SVG: Gráficos vetoriais escaláveis
  • Avif: Formato moderno de imagem com compressão eficiente com tamanho pequeno

Sobre o elemento img

A forma mais comum de inserir imagens no html é utilizando a tag img

trabalhando com links externos

o atributo src contem uma URL que aponta para a imagem que você deseja exibir. O atributo alt é usado para fornecer um texto alternativo que descreve a imagem, caso ela não possa ser exibida por algum motivo (por exemplo, se a imagem não for encontrada ou se o usuário estiver usando um leitor de tela).

SRC = Atributo responsável por especificar o caminho da imagem, ou seja, define onde a imagem está localizada.

Trabalhando com recursos locais

Para usar uma imagem local, ou seja, uma imagem que está salva no seu computador, você precisa fornecer o caminho correto para a imagem no atributo src.

ALT(alternative/alternativo) - É o texto alternativo que será apresentado caso a imagem não carregue

O atributo alt é Muito importante:

  • acessibilidade
  • Melhora a acessibilidade para pessoas com deficiência visual
  • Leitores de tela
  • Fornece contexto para mecanismos de busca
  • SEO
  • Ajuda a identificar o conteúdo da imagem quando ela não pode ser carregada
  • Fallback

Ajustando o tamanho das imagens

HEIGHT = O atributo serve para definir a altura da imagem

WIDTH = O atributo serve para definir a largura da imagem

Trabalhando com imagens responsivas

O picture é um elemento do HTML usado para imagens responsivas e adaptativas.

Essa tag, permite o seguinte:

  • Trocar a imagem com base no tamanho da tela
  • Melhorar a performance
  • Carregar imagens diferentes para mobile, tablet e desktop

srcset - Serve para oferecer múltiplas versões de uma imagem para diferentes tamanhos de tela e deixa o navegador escolher a melhor conforme o tamanho da tela, formato suportado, largura disponivel e performance do usuário.
