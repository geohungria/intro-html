Conhecendo o Formulários

Form

No HTML, a tag <form> é utilizada para criar formulários. Ele serve para coletar dados do usuario e enviar para um servidor para processamento.(Aplicação web). <form> é o contêiner principal para os elementos de formulário, como campos de entrada, botões, caixas de seleção, etc. Ele define a estrutura do formulário e especifica como os dados serão enviados para o servidor.
