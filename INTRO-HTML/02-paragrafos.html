Elemento blockquote

Esse é um exemplo de citação em bloco. Ele é usado para citar um trecho de texto mais longo, como um parágrafo inteiro ou mais. Ele geralmente é usado para destacar uma citação importante ou para citar um trecho de um texto que merece destaque.
